World Wide Web

The three most essencial technologys are: network adress, hypertext and hypermedia

all resources are adressed through the URL (the Uniform Resource Locator)

the URLs general component is: a URI (Uniform Resource Identifier)

Web Server

a Web Server is server software, or hardware dedicated to running this software, that can satisfy client requests on the World Wide Web

it can, in general, contain one or more websites, The primary function of a web server is to store, process and deliver web pages to clients.

it processes incoming network requests over HTTP and several other related protocols.

Web Client

a Web Client is is a software application for accessing information on the World Wide Web

When a user requests a web page from a particular website, the web browser retrieves the necessary content from a web server and then displays the page on the user's device.

Web Browser and Web Client are the same

HTML body Tag

Definition and Usage

defines "body"

contains all the Documents information. eg.: Headings, paragraphs; images...

HTML em Tag

Is used to define emphasized text

HTML Head Element

Cotains metadata (data about data)

Metadata contains: Document Title, character set, styles, scripts...

Metadata is not displayed

HTML html Tag

defines the root of the document

contains all other HTML elements (eccept "!DOCTYPE html" tag)

you should ALWAYS include the LANG (language) attribute inside the html Tag
